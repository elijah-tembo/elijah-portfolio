Here is what we are building:

# Portfolio Structure

Your website will have:

1. Hero Section (Your Photo + Introduction)
2. About Section
3. Services Section
4. Portfolio Section
5. Skills Section
6. Contact Section
7. Footer

Style:

* Dark Professional Theme
* Clean Modern Design
* Smooth Animations
* Fully Responsive

---

# Step 1 — Create Your Project Folder

Create this folder structure:

```
portfolio/
│
├── index.html
├── style.css
├── script.js
└── images/
```

This helps keep your project **clean and professional**.

---

# Step 2 — Create Your HTML File


<!DOCTYPE html>
<!-- This tells the browser we are using HTML5 -->

<html lang="en">
<!-- Root element of the website, lang="en" means English -->

<head>

    <meta charset="UTF-8">
    <!-- Allows website to support all characters -->

    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- Makes website responsive on mobile devices -->

    <title>Elijah Z Tembo | Portfolio</title>
    <!-- This is the title shown in browser tab -->

    <link rel="stylesheet" href="style.css">
    <!-- This links your CSS file to HTML -->

</head>

<body>
<!-- Everything visible on the website goes inside body -->

    <!-- Header Section (Top Navigation Bar) -->
    <header>

        <!-- Navigation Bar -->
        <nav>

            <!-- Logo / Name -->
            <h2 class="logo">Elijah Z Tembo</h2>

            <!-- Navigation Menu -->
            <ul>

                <!-- Menu Item -->
                <li><a href="#">Home</a></li>

                <!-- Menu Item -->
                <li><a href="#">About</a></li>

                <!-- Menu Item -->
                <li><a href="#">Services</a></li>

                <!-- Menu Item -->
                <li><a href="#">Portfolio</a></li>

                <!-- Menu Item -->
                <li><a href="#">Contact</a></li>

            </ul>

        </nav>

    </header>

</body>

</html>


Step 2 — style.css (With Comments)

Create style.css and paste this:

/* Reset default browser styles */
*{
    margin:0;
    padding:0;
    box-sizing:border-box;
    font-family: Arial, Helvetica, sans-serif;
}

/* Body styling */
body{
    background:#0b0b0b; /* Dark background */
    color:white; /* White text */
}

/* Header styling */
header{
    padding:20px 10%; /* Top/bottom and left/right spacing */
}

/* Navigation styling */
nav{
    display:flex; /* Places items in one row */
    justify-content:space-between; /* Space between logo and menu */
    align-items:center; /* Align vertically */
}

/* Menu list styling */
ul{
    display:flex; /* Horizontal menu */
    list-style:none; /* Remove bullets */
}

/* Menu items spacing */
ul li{
    margin-left:30px;
}

/* Menu links styling */
ul li a{
    text-decoration:none; /* Remove underline */
    color:white; /* Text color */
    font-weight:500;
}

/* Logo styling */
.logo{
    font-size:22px;
    font-weight:bold;
}
